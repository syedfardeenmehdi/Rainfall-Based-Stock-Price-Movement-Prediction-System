# 📈 Rainfall-Based Stock Price Movement Prediction System

## 📌 Overview
This project focuses on predicting stock price movements using rainfall data as an external influencing factor. It combines financial data with environmental data to analyze patterns and improve prediction accuracy using machine learning techniques.

---

## 🚀 Features
- Stock price trend prediction  
- Integration of rainfall data with stock data  
- Data preprocessing and feature engineering  
- Machine Learning model (Random Forest)  
- Performance evaluation and analysis  

---

## 🛠️ Tech Stack
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib  

---

## 📊 Dataset
- Historical stock price data  
- Rainfall dataset  
- Merged dataset with engineered features (lag features, seasonal indicators)  

---

## ⚙️ Methodology
1. Data collection (stock + rainfall data)  
2. Data cleaning and preprocessing  
3. Feature engineering (lag values, time-based features)  
4. Model training using Random Forest  
5. Model evaluation and prediction  

---

## ▶️ How to Run
1. Install required libraries:pip install pandas numpy scikit-learn matplotlib
2. 2. Run the training script:python train_rf_with_features.py
3. Analyze predictions and results  

---

## 📈 Results
- Improved prediction accuracy using rainfall-based features  
- Demonstrates impact of environmental factors on stock trends  

---

## 🎯 Purpose
To explore how external environmental factors like rainfall can influence financial markets and improve stock prediction models.
